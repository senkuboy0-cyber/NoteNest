# NoteNest

**Your thoughts, organized.**

A simple and elegant Android todo/notes app built with Kotlin, Room Database, MVVM architecture, and Material Design 3.

## Features

- Add, edit, delete notes
- Mark tasks as complete
- Search notes by keyword
- Pin important notes
- Sort notes by date, priority, alphabetical order
- Swipe to delete with undo
- Dark mode support
- Reminder notifications
